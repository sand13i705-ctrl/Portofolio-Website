Tentang Saya

Halo! Nama saya sendi. Saya adalah seorang mahasiswa, dengan pengalaman pernah membuat suatu web site stand bazar agar pembali bisa lebih nyaman untuk membeli.

Saya bersemangat dalam sebuag bisnis yang menguntungkan dan punya nilai profit tinggi. Saya suka memberikan ide ide yang kreatif dan menarik yang dapat membuat orang tertarik. Portofolio ini adalah tempat saya menunjukkan beberapa dokumentasi saya akhir akhir ini.

Di luar pekerjaan, saya menikmati dimana saya bisa mendapatkan uang lebih banyak dengan beberapa ide usaha saya. Saya percaya bahwa keseimbangan antara pekerjaan dan hobi adalah kunci untuk terus berkreasi dan inovatif.
